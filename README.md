# MusicTheory
**EN | [PL](#pl)**

A simple web I'm building to learn both **music theory** and HTML & JS.
You choose a scale, the app shows the notes and basic chords.
Work in progress - Project made for fun :)

---

## How to run
```bash
git clone https://github.com/mlodymrt/musictheory
open index.html
```

---

## Features (3.12.2025)
- show notes inside scales
- generate simple chords

---

## Plans
- an actual ui
- chord progressions and genres
