## Informacje:
Wpadłem na ten pomysł jakoś 10 minut temu na zajęciach zawodowych.
Główna idea to aplikacja webowa, która pomaga Ci pisać muzykę bez uczenia się teorii muzyki.
Zamysł jest taki, żeby stronka pozwalała użytkownikowi wybrać skalę, po czym pokazuje nuty i akordy w tej skali.
W przyszłości dodam opcję żeby wybrać konkretny gatunek muzyczny, na co apka wyrzuci popularne progresje akordów w tym gatunku.
Nie widziałem jeszcze czegoś w tym stylu, ale to pewnie dlatego że jestem ignorantem i zbytnio nie szukałem/
###Nie chcę aby to było narzędzie, które niczego nie uczy
Dla mnie nauka teorii muzyki jest dosyć ciężka i myślę, że taki narzędzie mogłoby pomóc nie tylko w samym pisaniu ale szybszej nauce.
Kiedy apka będzie pokazywać Ci akordy, zrobię tak aby pokazywało z czego się składają *(np. Cmaj = [C,E,G])* i w jaki sposób zostały stworzone *(np. major składa się z prymy, prymy+4, prymy+7)*.
Akordy które się tam znajdą to Major, Minor, D7, sus2 i sus4.

## Information:
So I came up with the idea for this project about 10 minutes ago in my cs class as I'm writing this.
The main idea is for this to be a web app that helps you write music without learning music theory.
I want the site to let you pick between scales, then show notes and chords in that scale. 
I also want to add an option to select the genre of music that you're interested in and show popular chord progressions in the genre of choice.
I don't think I've seen a website/app like this before but that might be because I haven't really looked.
###I don't want the project to be a tool that doesn't teach.
I find learning music theory pretty hard and I think that having a tool like this might help you learn it faster.
When the app is gonna show you chords I'll make it so that it shows what sounds they're made out of *(for example: Cmaj = [C,E,G])* while also showing how they were made *(for example a major triad consists of the root note, root+4 and root+7)*.
The chords that will be added are Major, Minor, D7, sus2 and sus4.
